# Git

### Configure setup
`git config`

### Start a repo
`git init`

### Stage a file
`git add git.md`

### Commit with a message
`git commit -m 'initial commit'`

### Add remote site
`git remote add origin https://github.com/diogorau/cheatsheets.gif`

### Create main branch
`git branch -M main`

### Push changes
`git push -u origin main`

### Pull (refresh) changes
`git pull origin main`
